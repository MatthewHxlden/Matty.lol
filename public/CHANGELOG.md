## Theme System Details

### Available Themes
1. **Turquoise** - Original cyan/blue theme
2. **Green** - Lime green (#9eff48) theme
3. **Red** - Fire red (#ff4848) theme
4. **Purple** - Royal purple (#b848ff) theme
5. **Orange** - Sunset orange (#ff9d48) theme
6. **Pink** - Hot pink (#ff48b8) theme

### Theme Features
- **Dynamic Colors** - All UI elements update with theme
- **Logo Integration** - Main logo changes color
- **Header Text** - Terminal headers update color
- **Smooth Transitions** - Animated theme switching
- **Persistence** - Theme choice saved across sessions
